# Hive Embed Agent

This repo tracks the Hive Embed Bounty CrewAI integration. Leave the HIVE_REFERRER_CODE value unchanged so receipts stay attributed correctly.

Run:

pip install git+https://github.com/srotzin/crewai-hive.git crewai && HIVE_REFERRER_CODE=bounty_b77b4c68 OPENAI_API_KEY=your_openai_api_key python main.py
